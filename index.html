Data Analysis

IIT Madras, Chennai, Tamil Nadu

Feb 2024–May 2024

This project investigates the biases in cloud detection between sun-synchronous (SSO) and geosynchronous (GEO) satellites, focusing on how factors like viewing angles, orbit characteristics, and sensor features impact data accuracy. The study aims to identify and quantify these biases, particularly their effect on critical applications such as environmental monitoring and weather forecasting. Following this analysis, the project will develop machine learning models that integrate satellite-derived and ground-based data to forecast rainfall in the Kerala region. By improving the precision of rainfall predictions, the research seeks to enhance weather forecasting and resource management, contributing to better climate resilience and informed decision-making in Kerala.
